PayIOTA plugin for Magento 1.x . 
Copyright 2017 PayIOTA.me.

Based on the CoinPayments.net's Magento 1.x plugin.

Installation
1. Find your Magento root directory (you will see the folders app, downloader, includes, media, etc.)
2. Upload all of the files in the included app folder into your Magento app folder.
3. In your Magento Configuration page, enable the PayIOTA.me payment method and fill in the configuration fields.
